## 一、项目概述

**CompanyHub** 是一套面向富钢公司内部管理创新和技术改造项目的 **全生命周期可视化管理平台**，支持从项目立项、过程汇报、文件归档到最终结项的全流程跟踪与调度。

口号：**"富于学，钢于行"**

### 核心能力

| 能力模块       | 说明                                                                 |
| -------------- | -------------------------------------------------------------------- |
| 项目看板       | 按状态分区展示所有项目（实施中 / 已完成 / 已结项 / 暂停中）           |
| 多维搜索与筛选 | 支持关键词搜索 + 标签多选过滤                                         |
| Excel 批量导入 | 从公司年度汇总表（.xlsx）一键增量导入，自动解析编号、部门、负责人、标签 |
| 文件资料管理   | 按项目分类存储立项申请表 / 结项PPT / 自由资料，支持单文件与批量ZIP下载 |
| 周期更新记录   | 里程碑式推进记录（可附图、附件），形成时间轴                           |
| 固化汇报日志   | 不可篡改的项目进展快照，支持追加备注                                   |
| 延期历史追踪   | 每次变更结项时间时自动存档原因，形成完整变更记录                       |
| 二维码权限沙盒 | 为任意项目生成 7 天有效期 JWT Token，可外部只读访问                    |
| 调度中枢       | Scheduler 页面展示待处理 AI 行动与全站历史动态                         |
| AI 配置中心    | 支持接入通义千问（兼容 OpenAI 规范）进行智能调度辅助                   |
